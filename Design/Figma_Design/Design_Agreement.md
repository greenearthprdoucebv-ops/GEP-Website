# Design agreement — GreenEarth Produce

Our team uses this so we all design the same way (colours, fonts, how round the buttons are, etc.). If Figma has different numbers, we update this file to match Figma.

The **WCAG checklist** is a different file — that one is only about accessibility, not about "our look".

Screenshot of our palette: [MoodboardV2.1.png](brand_palette_reference.png)

---

## Our 4 colours (from the moodboard)

- **Forest green** `#1B6B45` — main buttons, important links, primary actions
- **Leaf green** `#6DBE45` — accents, highlights, secondary elements
- **Cream** `#F5F2E8` — page background, soft fills
- **White** `#FFFFFF` — cards, modals, clean space

### Supporting neutrals

- **Charcoal** `#2C2C2C` — main text, headings
- **Warm grey** `#6B6B60` — softer text, small labels, captions
- **Ginger gold** `#C8A44E` — small pops of warmth (badges, rating stars, sale tags)

---

## How we actually use them on the site

- Main button: forest green `#1B6B45`, white text, hover a bit darker `#155A38`
- Second button style: leaf green `#6DBE45`, charcoal text `#2C2C2C`, hover `#5EAD38`
- Outline button: 2px border forest green, forest green text, background see-through
- Page background: cream `#F5F2E8`
- Cards / modals: white `#FFFFFF`
- Borders / input lines: light `#D6D3C8`
- Errors: red `#B42318` but we always show words + icon too, not only red
- Success: forest green `#1B6B45` (already our brand, works great for "added to cart" etc.)

I should double-check contrast (especially white on the coloured buttons) with a checker before we hand in — same as the WCAG file says.

## Fonts

We're using **DM Sans** for now (or whatever we locked in Figma).

- H1 **32px**, H2 **24px**, H3 **20px** — semibold, colour `#2C2C2C`
- Normal text **16px**, regular, `#2C2C2C`, line height ~1.5
- Small text **14px**, `#6B6B60`
- Button text **15px**, medium

---

## Spacing (rough guide)

4 / 8 / 16 / 24 / 32 / 48 px — we pick from this list so gaps don't look random.
Usually **16** between blocks, **24–32** for section padding.

---

## Rounded corners

- Buttons + text fields: **10px**
- Product cards: **12px**
- Pop-up windows: **16px**
- Photos with rounded corners: **8px**

---

## Buttons & fields

- Button height at least **44px** (good for phone)
- Padding left/right **20px** (or **24px** for big "shop now" style)
- Gap between icon and text **8px**
- Input height **44px**, same **10px** corners, **14px** padding inside, thin border `#D6D3C8`
- When you tab to a field, visible focus ring (we use forest green-ish so it matches)

---

## Screen sizes

- From **768px** wide → we can do 2 columns where it makes sense
- From **1024px** → desktop layout, content not wider than **1200px** in the middle
- Mobile: one column, **16px** padding on the sides

---

## Extra bits

- Card shadow (soft): `0 2px 8px rgba(44, 44, 44, 0.06)`
- Bigger shadow for modals: `0 8px 24px rgba(44, 44, 44, 0.10)`
- Icons: **20px** normally, **24px** in headers
- Leave **16px** empty space around the logo
- Photography style: warm, natural light — think farm-to-table, hands in the soil, fresh produce close-ups (like the ginger moodboard)
